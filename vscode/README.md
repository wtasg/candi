# Candi Theme for VS Code

Scandinavian design theme based on the Candi Design System.

## Included Themes

- **Candi Light**
- **Candi Dark**

## Installation

1. Open VS Code and the Extensions view (`Ctrl+Shift+X`).
2. Click the "..." menu and select **Install from VSIX...**.
3. Choose the `.vsix` file in this directory.

## Activation

1. Go to **File > Preferences > Theme > Color Theme**.
2. Select **Candi Light** or **Candi Dark**.
