# Candi Theme for VS Code

Scandinavian design theme based on the Candi Design System.

## Included Themes

- **Candi Light**
- **Candi Dark**

## Features

- 285+ UI color definitions
- 100+ token scopes for syntax highlighting
- Semantic highlighting for TypeScript, Python, Rust, Go
- Language-specific syntax for HTML, CSS, JS/TS, JSX/TSX, Python, Go, Rust, and more
- Rainbow bracket pair colorization using primitive colors
- Complete dark/light mode support

## Primitive Colors

The theme includes 10 primitive color families used for bracket highlights:

| Color   | Usage                          |
|---------|--------------------------------|
| Blue    | Bracket level 1                |
| Magenta | Bracket level 2                |
| Cyan    | Bracket level 3                |
| Yellow  | Bracket level 4                |
| Green   | Bracket level 5                |
| Red     | Bracket level 6                |

See [COLOR_REFERENCE.md](COLOR_REFERENCE.md) for all available color tokens.

## Installation

1. Open VS Code and the Extensions view (`Ctrl+Shift+X`).
2. Click the "..." menu and select **Install from VSIX...**.
3. Choose the `.vsix` file in this directory.

## Activation

1. Go to **File > Preferences > Theme > Color Theme**.
2. Select **Candi Light** or **Candi Dark**.
