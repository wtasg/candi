# Changelog

## 0.0.26

### Added

- **Initial Release**: Flutter-based showcase application for the Candi Design System.
- **Gallery Screen**: Display of all color tokens across various Material 3 components.
- **Playground Screen**: Interactive theme exploration with:
  - Accessibility contrast checks.
  - Color vision deficiency simulations (Protanopia, Deuteranopia, Tritanopia, Achromatopsia).
  - Light/Dark mode toggling.
- **Theme Provider**: State management for dynamic theme switching.
- **Responsive Layout**: Optimized for Web and Desktop platforms.
