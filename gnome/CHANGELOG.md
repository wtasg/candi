# Changelog

All notable changes to the Candi GNOME theme will be documented in this file.

## [0.0.17] - 2025-12-26

### Added

- Complete GTK3 and GTK4 theme coverage
- Automatic light/dark mode switching support
- Enhanced widget styling for modern GNOME applications

### Changed

- Improved color contrast for better accessibility
- Updated color values to match Candi Design System v0.0.17
- Refined button and input field styling

## [0.0.12] - Previous Release

### Added

- Initial GNOME GTK3/GTK4 themes
- Light and dark variants
- Basic widget coverage
- Theme metadata (index.theme)
