# Changelog

## 0.0.36

### Added

- **Focus Indicators**: `*:focus-visible` with accent outline for accessibility
- **Dialog Styling**: Message dialogs with surface background, rounded corners, shadows
- **Calendar**: Date picker with header, selected, highlight, and indeterminate states
- **Treeview**: Selection, hover, header column styling
- **Libadwaita Compatibility**: 30+ `@define-color` entries (accent, destructive, window, headerbar, card, dialog, popover, sidebar, etc.)

### Changed

- **Palette**: Warm-gray neutrals, brighter light mode
- **Disabled States**: Improved opacity handling
- **Backdrop**: Adjusted from 0.8 to 0.9 opacity

## 0.0.32

- **Synchronization**: Version bump to align with core design system v0.0.32.
- **Refinement**: Minor updates to color palette values.

## 0.0.31

- **Synchronization**: Version bump.
